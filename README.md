# Импортозамещение или локализация - Круглый стол

Адаптивный лендинг для мероприятия "Импортозамещение или локализация" на выставке "Связь-2026".

## 🎨 Дизайн

- **Пиксель-перфектный** макет, точно следующий оригинальному дизайну Figma
- **Темная тема** с градиентами голубого цвета (#75C9EA, #0BA1DD)
- **Шрифт**: Montserrat (Regular, Medium, SemiBold, Bold)
- **Цветовая схема**: Темно-синий фон (#000C14) с акцентами голубого

## ✨ Особенности

### Анимации
- **Плавные появления** элементов при прокрутке с использованием Motion (Framer Motion)
- **Параллакс эффекты** для изображений
- **Анимированные частицы** на фоне
- **Плавающие элементы** для создания глубины
- **Пульсирующие градиенты** на фоновых SVG
- **Интерактивные hover эффекты** на всех кнопках и карточках
- **Прогресс-бар прокрутки** страницы

### Адаптивность
- **Полностью адаптивный** дизайн для всех устройств
- **Мобильные точки останова**: 
  - Mobile: < 768px
  - Tablet: 768px - 1024px
  - Desktop: > 1024px
- **Адаптивная навигация** с мобильным меню
- **Гибкая сетка** для всех компонентов
- **Оптимизированные изображения** для разных размеров экранов

### Компоненты

1. **Header** - Фиксированный хедер с навигацией и мобильным меню
2. **Hero** - Главная секция с заголовком и CTA-кнопками
3. **Features** - Карточки с ключевыми особенностями
4. **StatsSection** - Анимированные счетчики статистики
5. **ProgramTabs** - Интерактивные вкладки с программой по дням
6. **RoundTableSection** - Программа круглого стола с темами
7. **LectoriumSection** - Расписание лектория и нетворкинга
8. **TestimonialsSection** - Отзывы участников
9. **PartnersSection** - Логотипы партнеров
10. **RegistrationForm** - Форма регистрации на мероприятие
11. **Footer** - Подвал с контактами и ссылками

### Интерактивность

- **Scroll to Top** - Кнопка быстрого возврата наверх
- **Мобильное меню** - Слайдящее боковое меню для мобильных устройств
- **Форма регистрации** - Валидация и отправка данных
- **Интерактивные карточки** - Hover эффекты с трансформацией
- **Плавная прокрутка** - Smooth scroll для якорных ссылок

## 🚀 Технологии

- **React 18** - Библиотека для создания UI
- **TypeScript** - Типизация
- **Tailwind CSS v4** - Утилитарный CSS фреймворк
- **Motion (Framer Motion)** - Библиотека анимаций
- **Vite** - Быстрая сборка

## 📱 Адаптивность

Приложение полностью адаптивно и отлично работает на:

- 📱 Мобильных телефонах (320px - 767px)
- 📱 Планшетах (768px - 1023px)
- 💻 Десктопах (1024px+)
- 🖥️ Больших экранах (1440px+)

## 🎯 Производительность

- ⚡ Оптимизированные анимации (GPU-ускорение)
- 🖼️ Lazy loading для изображений
- 📦 Code splitting для компонентов
- 🎨 CSS-in-JS для стилизации с минимальным размером бандла

## 💡 Ключевые решения

- **Модульная архитектура** - Каждый компонент независим и переиспользуем
- **Анимации на входе** - Все секции появляются с анимацией при скролле
- **Производительность** - Использование `viewport` в Motion для оптимизации
- **Доступность** - ARIA-атрибуты для всех интерактивных элементов
- **Семантический HTML** - Правильная структура для SEO
